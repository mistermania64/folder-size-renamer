# folder-size-renamer
Helper script used to batch rename folders in a directory by appending their size to the end of their name.

Usage: 
1. Place `fileRename.py` in the directory you want to batch rename.
2. Open Command Prompt (CMD) and run the script with `python fileRename.py`.
3. Any folders in this directory will automatically be renamed with their filesize added to the end of their original name.

Note: Use the `fileRenameStrip.py` script in the same manner to remove the appended file size from each folder's name.
